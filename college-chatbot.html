Project Overview

What is the Chatbot?

🎯
The Problem
Traditional college websites provide information through static pages and PDFs, requiring manual navigation. During admission periods, colleges receive many repetitive queries about courses, fees, hostels, and scholarships — increasing administrative workload and delaying responses.
💡
Our Solution
An AI-based chatbot that acts as a virtual assistant — interacting in natural language to provide instant answers about admissions, courses, fees, hostel, placements, scholarships, and more. Operates 24×7 without human intervention.
🏛
Institution

Category
Artificial Intelligence / Web Application. Built using Python and Flask for backend, HTML5, CSS3, JavaScript for frontend, NLTK & Scikit-learn for NLP, and a JSON dataset as the knowledge base.

Processing Pipeline

How It Works

1
User Sends a Query
The user types a question in the chat interface (e.g. "What are the BCA fees?" or "Is hostel available?"). The frontend sends this to the Flask /chat API endpoint via an HTTP POST request.
2
Query Preprocessing
The text is converted to lowercase, special characters removed, and tokenized using NLTK. Stop-words are eliminated and lemmatization converts words to their base forms — improving matching accuracy.
3
TF-IDF Vectorization
The preprocessed query is transformed into a TF-IDF vector using Scikit-learn's TfidfVectorizer, which is then compared against the pre-built corpus of all intent patterns from the JSON knowledge base.
4
Cosine Similarity — Intent Detection
Cosine similarity scores are calculated between the query vector and all pattern vectors. The intent with the highest similarity score (above threshold 0.25) is selected as the predicted intent category.
5
Response Generation & Display
A suitable response is randomly selected from the matched intent's responses array in the JSON knowledge base. If no intent matches, a fallback response is returned. The response is sent to the frontend and displayed instantly.
System Design

System Architecture

Application Layer
College Enquiry Chatbot Interface
The top layer where users interact through the Web/App Chat Window. Built with HTML5, CSS3, and JavaScript — sends queries to the Flask API and displays chatbot responses dynamically.
Presentation Layer
User Interface & Input Handling
Displays messages to the user and accepts user queries (fees, courses, admissions, etc.). Responsible for rendering the chat bubbles, quick topic buttons, typing indicators, and timestamps.
Processing Layer
NLP / Query Processing Engine
The intelligence layer — handles tokenization, stop-word removal, lemmatization (NLTK), TF-IDF vectorization, cosine similarity intent matching, and response selection using Python & Scikit-learn.
Data Layer
JSON File Storage — Knowledge Base
Stores all college information — courses, fees, admission details, hostel, placements — as structured intents (tag, patterns, responses) in intents.json. Chatbot reads and retrieves data directly from this file.
Modular Design

System Modules

Module 01
User Interaction Module
  • Chat interface (browser-based)
  • Input handler — captures user queries
  • Response display — renders bot replies
  • Quick topic buttons for easy navigation
Module 02
Query Processing Module
  • Lowercase conversion
  • Special character removal
  • Tokenization of input text
  • Stop-word removal for clean input
Module 03
NLP Processing Module
  • Lemmatization using NLTK WordNetLemmatizer
  • Keyword extraction from user queries
  • Word normalization for better matching
  • Preprocessing pipeline using NLTK libraries
Module 04
Intent Detection Module
  • TF-IDF vectorization (Scikit-learn)
  • Cosine similarity scoring
  • Threshold-based intent classification (0.25)
  • Best-match intent selection
Module 05
Response Generation Module
  • Response retrieval from JSON knowledge base
  • Random response selection from pool
  • Response formatting for display
  • Fallback handling for unknown queries
Module 06
Knowledge Base Module
  • intents.json — primary data store
  • 14+ intent categories covering college info
  • Tag, patterns, and responses structure
  • Easy manual updating and maintenance

Coverage

Scope of the Application

Admissions & Courses — Handles queries for all UG (BA, BSc, BCom, BBA, BCA, BSW) and PG (MSc, MCom, MA, MSW) programmes including eligibility, duration, fees, and specializations.
Facilities & Infrastructure — Answers questions about library (55,000+ books), science labs, computer labs, ICT-enabled smart classrooms, auditorium, gymnasium, and Wi-Fi campus.
Hostel & Campus Life — Provides details on separate boys & girls hostels with Wi-Fi, CCTV, purified water, steam cooking, and sports grounds (3 large grounds).
Scholarships & Financial Aid — Covers merit-based scholarships, government schemes (SC/ST/OBC Post-Matric), minority scholarships, and sports quota fee concessions.
Placements & Career — Shares placement cell overview, NIRF 2025 data (53 students placed, median ₹2.52 LPA), soft skills programs, and campus recruitment drives.
Future Enhancements — Planned expansions include multilingual support, voice-based interaction, speech recognition, mobile app (Android/iOS), WhatsApp/Telegram integration, and deep learning models.
